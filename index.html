السعر: جنيه

- صفحة بسيطة فيها إدخال كود الدخول ("1111") - لو صح، يدخل على dashboard.html - تسجيل الدخول بكود - التحكم في: - رقم فودافون كاش - ID حساب الديسكورد - السعر - الصورة - عرض الطلبات: - حالة الطلب: Pending / Accepted / Rejected - أزرار: قبول - رفض { "vodafone_number": "010xxxxxxxx", "discord_id": "123456789012345678", "price": "150", "image": "images/boost14.png" } - تحميل البيانات من settings.json - عرض الطلبات من orders.json - التحكم في الإعدادات - حفظ الطلبات بعد القبول / الرفض - يقرأ tokens.txt - يفتح كل توكن باستخدام Puppeteer - يحاول يدخل السيرفر اللي العميل كتبه - يتوقف عند ظهور Captcha - بعد حل الكابتشا، يعمل Transfer للبوستات من السيرفر القديم إلى الجديد - استخدام Bot API - عند استلام طلب جديد: إرسال رسالة فيها كل تفاصيل الطلب إلى @boostwave0 - يتم التحقق يدويًا من الدفع (Vodafone أو Discord Credit) - العميل يكتب رقم التحويل أو يرفع لقطة شاشة (في مرحلة لاحقة نقدر نربط API) - عند الضغط على "قبول" في dashboard → يتم استدعاء boost.js تلقائيًا ويبدأ التنفيذ
